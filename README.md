# sales-management-system-ds
# Sistema de Administración de Ventas con Inteligencia de Negocios

Este proyecto es una solución integral para la gestión de inventarios, ventas a crédito y análisis financiero, diseñado para usuarios con conocimientos técnicos mínimos pero con una arquitectura robusta para Ciencia de Datos.

## Características Principales
- **Gestión de Inventario:** Soporte para códigos de barras y códigos alfanuméricos vía OCR.
- **Módulo Financiero:** Cálculo de Utilidad Neta considerando Gastos Operativos.
- **Sistema de Abonos:** Semáforo de pagos y recordatorios automáticos vía WhatsApp.
- **Arquitectura:** Basada en Google Cloud, AppSheet y Google Apps Script.
- **Preparado para ML/DL:** Estructura de datos optimizada para modelos de riesgo de impago y demanda estacional.
